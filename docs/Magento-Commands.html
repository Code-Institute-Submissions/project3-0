How to use the Magento command-line interface (CLI)

The Magento CLI enables you to run and automate common installation and configuration tasks from the command line.
Magento tasks are executed by shell commands with /bin/magento.

After install you need to set the permission on the Magento CLI.
chmod 744 /bin/magento
Then ensure you are using the correct version of php in cli mode, this is the command.
 
/usr/local/php71/bin/php-cli bin/magento
Other commands that need to be used are to flush the cache:
 
/usr/local/php71/bin/php-cli bin/magento cache:flush

the full list of commands see below.

To get the javscript demo working I needed to run these command 

/usr/local/php71/bin/php-cli bin/magento setup:upgrade

bin/magento setup:upgrade

bin/magento cache:clean

bin/magento indexer:reindex


also see https://devdocs.magento.com/guides/v2.3/install-gde/install/cli/install-cli-subcommands.html
