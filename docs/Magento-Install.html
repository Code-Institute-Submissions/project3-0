How to use install Magento

To install Magento you can download an open source version from the download section of the website.
The application is also available on many hosting control Panel's and this process usually creates the relevant MySQL database for you.

I installed  Magento at : https://ciaranquinlan.com
The Administrative URL : https://ciaranquinlan.com/index.php/admincjq

In order to use Magento for the project I needed to do a few tasks to administer Magento
  1. Enable SSH access.
    To do a lot of tasks its easier if you can log into the Magento server and execute shell commands.
    The details of how i went about setting up ssh is here.
  2. Magento CLI
    I need to learn some commands to perform admin and maintenance tasks on the Magento app and DB
  3. composer
    Composer is a tool for dependency management in PHP. It allows you to declare the libraries your project depends on and it will manage (install/update) them for you.
    Magento uses composer to manage installs and upgrades. You need to add you Magento credentials from your magento marketplace account to use composer.
    Access the keys under products and copy the keys, you need to use the private and public keys. With these you can install from the Magento repo's 
    Public Key: 96f2ea65b1a9dab3812b464730d48e35 
Private Key: 0ae25fe32b01b1fa18ad5bb09f57052f
  4. Enable backups
    to ensure that I could roll back the system if I had problems. I needed to use the cli to enable backuos then i could use the system backup tool
    /usr/local/php71/bin/php-cli bin/magento config:set system/backup/functionality_enabled 1
  5. Flush cashe
    /usr/local/php71/bin/php-cli bin/magento cache:flush
  6. Set ownership and permissions for a one-user system:.
    To set permissions before you install the Magento software run these commands
    find var generated vendor pub/static pub/media app/etc -type f -exec chmod u+w {} +
    find var generated vendor pub/static pub/media app/etc -type d -exec chmod u+w {} + chmod u+x bin/magento
  7. Set the Magento mode to developer
    First check the mode 
    /usr/local/php71/bin/php-cli bin/magento deploy:mode:show
    /usr/local/php71/bin/php-cli bin/magento deploy:mode:set
  8. Install sample data
    /usr/local/php71/bin/php-cli bin/magento sampledata:deploy
